# Манифест пакета TalentScan Applicant / Career Agent v1.2

## Назначение

Этот ZIP — база отдельного агента ТалентСкан для соискателей и людей, которые выбирают карьерный путь.

Агент помогает пользователю проверить:

- карту талантов и профессиональные направления;
- текущую роль;
- конкретную вакансию;
- несколько вакансий / вариантов между собой;
- рабочую среду;
- руководителя и рабочую связку;
- вопросы работодателю;
- план действий 7 / 30 / 90.

## Что изменено в v1.2

1. Активный дизайн и структурный эталон — Дарья, dark-card стиль.
2. Добавлены свежие эталоны: текущая роль котокафе, вакансия БАНКИРРО, простая совместимость Артур × Дарья, кабинет Дарьи v1.3.
2.1. Добавлен и назначен активным эталоном для `VACANCY_ASSESSMENT` файл Михаила: `EXAMPLE_MIKHAIL_ONESUMMER_VACANCY_ASSESSMENT_V2_SELF_PRESENTATION.html`. Этот эталон обязателен для всех будущих карточек соответствия вакансии и содержит блок “Как преподнести себя на собеседовании”.
3. Добавлены QA-протоколы для каждого типа карточки.
4. Добавлены режимы работы: разработка и тестовый режим.
5. Усилено правило независимости кейсов: новый бодиграф / новая роль / новая вакансия не наследуют прошлый контекст.
6. TeamScan очищен от старого большого multi-context compatibility pack. Активна только applicant-логика.
7. Соционика не активна в этой версии.
8. PDF — только по прямой просьбе.

## Формат по умолчанию

- В режиме разработки: сначала уточнять задачу, предлагать варианты, не генерировать разборы автоматически.

- PDF делать только по прямой просьбе пользователя.
