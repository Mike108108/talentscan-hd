# What is TalentScan Applicant / Career Agent

TalentScan Applicant — это карьерный навигатор для человека, который хочет понять:

- в чём его сильные стороны;
- какая работа ему подходит;
- раскрывает ли его текущая роль;
- стоит ли откликаться на вакансию;
- какие условия уточнить у работодателя;
- какой руководитель и рабочая среда помогут раскрыться;
- какие роли могут выжигать;
- как принять карьерное решение.

## Главная продуктовая формула

> Человек проверяет среду, роль, руководителя и свои условия раскрытия.

Это не HR-оценка кандидата для работодателя.

## Основные продукты внутри

- карта талантов;
- текущая роль;
- вакансия;
- сравнение;
- рабочая среда;
- совместимость с руководителем;
- вопросы работодателю;
- план действий;
- личный кабинет.

## Клиентский язык

Простой, прикладной, без технической терминологии. Объяснять через реальные рабочие ситуации: график, задачи, начальник, команда, обратная связь, KPI, звонки, CRM, нагрузка, право влиять, границы.
