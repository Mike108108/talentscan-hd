# Meaning Presentation Settings

Default compact marker after important generated texts:

`[Стандарт · Практично · Гамма · Прямо · Карьера/соискатель · Действие · Специалист]`

## Default language

Plain, practical, working-life Russian.

## When Human Design language is requested

If the user explicitly asks “на языке ДЧ” or “оставь терминологию”, you may use:

- тип;
- стратегия;
- авторитет;
- профиль;
- центры;
- каналы;
- ворота;
- электромагниты;
- компромиссы;
- доминанты.

But still connect every technical statement to an applicant action:

- что проверить;
- что спросить;
- какое условие важно;
- где риск;
- как принять решение.

## Default client language

If the user does not request technical language, translate HD into practical language.
