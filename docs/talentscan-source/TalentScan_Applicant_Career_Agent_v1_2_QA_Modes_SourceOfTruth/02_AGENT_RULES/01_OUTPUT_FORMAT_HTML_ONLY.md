# Output format rules

## Default

HTML standalone card is default output in Test Mode.

## PDF

PDF is not default. PDF only if user explicitly asks:

- “сделай PDF”;
- “выгрузи PDF”;
- “сделай эти страницы в формате PDF”;
- “нужен PDF для клиента”.

## Text

Text-only output only if user asks.

## Development mode

In Development mode, do not automatically create HTML/PDF until task and format are clear.

## HTML design

Use Darya dark style as active visual system.
Do not invent new design from scratch unless user explicitly asks.
