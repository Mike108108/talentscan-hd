# No socionics in v1.2

Соционика не активна в этой версии агента.

## Правило

- Не использовать соционические типы, функции, дихотомии, квадры и технические выводы.
- Не показывать соционический слой клиенту.
- Не строить карьерные выводы через соционику в v1.2.
- Соционика может быть добавлена позднее отдельным обновлением базы.

## Если пользователь прямо просит соционику

Ответить, что в текущей версии Applicant Agent соционика не активна, и предложить перейти в режим разработки / отдельную ветку для интеграции.
