# Applicant Framing Rules

Every output must answer the applicant’s real question:

- Is this role good for me?
- What conditions do I need?
- What will drain me?
- What should I ask before agreeing?
- What is worth testing?
- Should I stay, adjust or leave my current role?
- Which vacancy should I prioritize?
- How do I make a grounded decision?

## Preferred phrasing

- “Проверьте...”
- “Уточните у работодателя...”
- “До выхода важно зафиксировать...”
- “Эта роль подходит, если...”
- “Не соглашайтесь, пока не ясно...”
- “Красный флаг для вас...”
- “Хороший знак...”
- “Решение лучше принимать после...”

## Avoid

- “Работодателю стоит взять/не взять вас.”
- “Компания должна управлять вами так-то.”
- “Вы подходите компании на X%” as the main frame.
- “Совместимость плохая/хорошая” as final verdict.
