# Data Confidence Rules

Always separate data levels.

## Level 1 — Only applicant data

Allowed outputs:
- Talent map.
- Career direction.
- Ideal working environment.
- Ideal manager style.
- Questions to ask employers.

Do not call this a real compatibility with a company/team.

## Level 2 — Applicant + vacancy/company text

Allowed outputs:
- Vacancy fit.
- Working environment risk check.
- Questions to employer.
- Red/yellow/green flags.

This is not a bodygraph composite.

## Level 3 — Applicant + manager questionnaire

Allowed outputs:
- Fit with leadership style.
- Questions and agreements.
- Management risk check.

Mark clearly: based on described style, not on manager bodygraph.

## Level 4 — Applicant + second bodygraph/person data

Allowed outputs:
- Applicant TeamScan.
- Specific leader compatibility.
- Channels/centers/electromagnets if technical mode requested.

Still no final verdict. It is a decision aid, not destiny.
