# Input Examples

## Example 1

User sends only a bodygraph.

Output: Talent Map.

## Example 2

User says: “Я работаю администратором в котокафе, хочу понять подходит ли мне.”

Output: Current Role Analysis.

## Example 3

User uploads a vacancy.

Output: Vacancy Assessment.

## Example 4

User says: “Сравни текущую роль и вакансию куратора клиентского опыта.”

Output: Comparison Card.

## Example 5

User asks: “Какой руководитель мне подходит?”

Output: Ideal Manager / Working Environment Card.

## Example 6

User provides their bodygraph and a leader’s bodygraph.

Output: Applicant TeamScan.

Important: frame it as “how the applicant checks the leader and conditions,” not “how the leader should manage the applicant.”
