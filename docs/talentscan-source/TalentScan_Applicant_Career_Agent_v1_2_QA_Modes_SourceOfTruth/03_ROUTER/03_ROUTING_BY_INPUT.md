# Routing by input examples

## Input: only bodygraph

Output: Talent Map.

## Input: bodygraph + “я работаю администратором в котокафе...”

Output: Current Role.

## Input: bodygraph + vacancy text

Output: Vacancy Assessment.

## Input: “сравни текущую роль и эту вакансию”

Output: Comparison.

## Input: “какой руководитель мне подходит”

Output: Working Environment / Ideal Manager.

## Input: “вот мой бодиграф и бодиграф руководителя”

Output: Applicant Compatibility.

## Input: “сделай личный кабинет”

Output: Applicant Cabinet.

## Важно

Если пользователь просто прислал новый бодиграф в тестовом режиме — это новый человек, не продолжение Дарьи.
