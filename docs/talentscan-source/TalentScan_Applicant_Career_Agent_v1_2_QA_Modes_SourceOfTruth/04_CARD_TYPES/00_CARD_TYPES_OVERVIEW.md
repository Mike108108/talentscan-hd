# Card types overview v1.2

## Активные типы карточек

1. **TALENT_MAP** — карта талантов.
2. **CURRENT_ROLE** — анализ текущей роли.
3. **VACANCY_ASSESSMENT** — оценка рассматриваемой вакансии.
4. **COMPARISON** — сравнение ролей / вакансий / вариантов.
5. **WORKING_ENVIRONMENT** — подходящая рабочая среда.
6. **APPLICANT_COMPATIBILITY / TEAMSCAN** — руководитель / команда / компания через призму соискателя.
7. **QUESTIONS_TO_EMPLOYER** — вопросы на собеседовании.
8. **CAREER_PLAN_7_30_90** — план действий.
9. **APPLICANT_CABINET** — многоуровневый личный кабинет.

## Как отличать

- Есть только бодиграф/данные человека → `TALENT_MAP`.
- Человек уже работает и описал текущую работу → `CURRENT_ROLE`.
- Человек рассматривает новую вакансию → `VACANCY_ASSESSMENT`.
- Есть 2+ вариантов → `COMPARISON`.
- Вопрос про “какая среда мне подходит” → `WORKING_ENVIRONMENT`.
- Есть руководитель / второй бодиграф / стиль управления → `APPLICANT_COMPATIBILITY`.
- Нужно собрать UX приложения → `APPLICANT_CABINET`.

## Формат

HTML по умолчанию. PDF только по просьбе.
