# CURRENT_ROLE_ANALYSIS

## Purpose

Help the applicant decide what to do with their current role.

## Main question

> Should I stay, adjust conditions, grow inside this role, use it as a bridge, or prepare to leave?

## Mandatory blocks

1. Quick answer by current role.
2. Fit percentage for current conditions.
3. Fit percentage after improvements / negotiations.
4. What fits.
5. What drains.
6. Task matrix with percentages.
7. What to discuss with the employer/manager.
8. Red flags.
9. Practical tests for 1–4 weeks.
10. Decision: stay / adjust / temporary bridge / prepare change / leave.

## Applicant framing

Write:
- “что вам обсудить с руководством”;
- “что изменить в условиях”;
- “какие границы зафиксировать”;
- “как понять, что роль больше не ваша”.
