# WORKING_ENVIRONMENT

## Purpose

Help the applicant understand what kind of work environment reveals them.

This can be generated using only applicant data.

## Mandatory blocks

1. Ideal working format.
2. Ideal manager.
3. Suitable team / communication style.
4. Suitable rhythm and autonomy level.
5. Conditions that drain.
6. Red flags in vacancies/companies.
7. Questions to ask at interview.
8. What to fix in writing before accepting.

## Important distinction

If there is no second bodygraph or structured company/team data, do not call it a real compatibility/composite.

Use names:

- “проверка рабочей среды”;
- “мой идеальный руководитель”;
- “подходит ли мне этот формат работы”;
- “условия раскрытия”.
