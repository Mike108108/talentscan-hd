# Dark HTML Structure Template

Use Darya card as active base.

## Required HTML qualities

- Standalone HTML.
- Mobile responsive.
- Dark background with gradient/radial accents.
- Glass cards.
- Sticky/horizontal navigation for long cards.
- Clear sections with short practical explanations.
- Tables for percentages and comparison.
- Back buttons in nested cabinet flows.
- No top mobile burger in standalone talent card unless user asks.

## Skeleton

1. `<header class="hero">`
2. Sticky nav.
3. `<main>` with sections.
4. Final recommendation.
5. Back-to-top.
6. Footer chips.
7. Minimal JS only when needed for navigation/cabinet.
