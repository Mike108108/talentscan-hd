# Template — Applicant Talent Map

Use this order:

1. Hero: name + short formula.
2. Identity cards: best format, key talent, main risk.
3. Overall conclusion.
4. Metrics.
5. Talents in working translation.
6. Strengths.
7. Risks.
8. Suitable professional directions with percentages.
9. Controversial / low-fit directions with percentages.
10. Roles to check.
11. Working environment.
12. Practical tests.
13. Final recommendation.

Keep text concise inside HTML. Put long methodological explanations outside the card if needed.
