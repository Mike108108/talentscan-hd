# Template — Applicant Vacancy Assessment

1. Quick answer.
2. Fit percentage.
3. What fits.
4. What drains / risks.
5. What to ask employer.
6. Green/yellow/red conditions.
7. How to present yourself: позиционирование на собеседовании, что подчеркнуть, что не выпячивать, готовые фразы.
8. Whether to apply / test / avoid.

Use applicant language:

- “вам стоит уточнить”;
- “не соглашайтесь, пока...”;
- “эта вакансия подходит, если...”;
- “красный флаг для вас...”;
- “следующий шаг”.
