# Template — Applicant Vacancy Assessment Card

Использовать для вакансии, которую человек рассматривает.

## Обязательные блоки

1. Hero: имя + название вакансии + компания.
2. Сильный быстрый ответ: смотреть глубже / откликаться / пауза / не основной вектор.
3. Процент соответствия.
4. Данные вакансии: зарплата, график, формат, обязанности, требования.
5. Главные плюсы вакансии для человека.
6. Главные риски.
7. Матрица задач: зона / совпадение / почему / что проверить.
8. Что уточнить у работодателя:
   - задачи;
   - KPI;
   - график;
   - обучение;
   - руководитель;
   - связь вне работы;
   - эмоциональная нагрузка;
   - рост;
   - бонусы и скрытые продажи, если есть.
9. Зелёные, жёлтые и красные сигналы.
10. Как преподнести себя на собеседовании / как себя продать под вакансию:
   - главная позиция соискателя;
   - что подчеркнуть;
   - как говорить про опыт;
   - какие риски закрыть;
   - что не выпячивать;
   - готовые фразы и короткая самопрезентация.
11. Матрица решения после собеседования.
12. План: до отклика / на собеседовании / после собеседования.
13. Итоговая рекомендация.

## Главное

Вакансия “рассматривается”, а не оценивается окончательно. Нельзя писать финальный приговор без ответов работодателя.
