# Template — Applicant Cabinet UX

Личный кабинет соискателя должен быть многоуровневым, не плоским лендингом.

## Главные вкладки

1. Обзор.
2. Карта талантов.
3. Профориентация.
4. Моя текущая роль.
5. Вакансии.
6. Сравнение.
7. Рабочая среда.
8. Совместимости.
9. Интервью.
10. План.
11. Данные.

## UX-правила

- Быстрая карточка + кнопка “читать подробнее”.
- Вложенные экраны: список → детальная карточка.
- Кнопки “назад” на всех вложенных экранах.
- Кнопки “добавить” там, где пользователь может добавить данные.
- Кнопка “изменить” для текущей роли.
- Вакансии должны храниться списком.
- Совместимости должны храниться списком.
- Сравнение должно позволять выбирать варианты.
- Детальные карточки должны открываться без потери кабинета.
