# Template — Applicant Compatibility / Manager Fit, plain language

Использовать для соискателя, который проверяет конкретного руководителя или стиль управления.

## Обязательные блоки

1. Hero: человек × руководитель.
2. Быстрый вывод: можно рассматривать / нужна пауза / красные условия.
3. Что может быть сильным в связке.
4. Как принимать решение.
5. Что спросить у работодателя / руководителя.
6. Какие условия проверить.
7. Стиль руководителя: что помогает / что давит.
8. Какая роль рядом с этим руководителем подходит.
9. Риски.
10. Договорённости до “да”.
11. Матрица: да / пауза / нет.
12. Чек-лист перед согласием.
13. Итог.

## Язык

По умолчанию — простой язык, без технических терминов.
