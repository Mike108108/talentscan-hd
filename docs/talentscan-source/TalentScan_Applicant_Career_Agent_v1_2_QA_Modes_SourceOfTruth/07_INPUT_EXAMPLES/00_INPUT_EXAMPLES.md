# Input Examples — Applicant Agent v1.2

## 1. Только бодиграф

Пользователь присылает бодиграф без текста.

В тестовом режиме: создать HTML-карту талантов. Каждый новый бодиграф = новый человек.

## 2. Бодиграф + текущая роль

Пример:

> Я работаю администратором в котокафе. Вот мои данные/бодиграф. Проверь, насколько роль мне подходит.

Создать: Current Role Card.

## 3. Бодиграф + вакансия

Пример:

> Я рассматриваю вакансию менеджера по работе с клиентами. Вот текст вакансии.

Создать: Vacancy Assessment Card.

## 4. Сравнение

Пример:

> Сравни мою текущую роль и эту вакансию.

Создать: Comparison Card.

## 5. Руководитель / совместимость

Пример:

> Вот мои данные и данные руководителя. Посмотри, стоит ли мне идти к нему работать.

Создать: Applicant Compatibility Card.

## 6. Рабочая среда

Пример:

> Какая рабочая среда мне подходит?

Создать: Working Environment Card.

## 7. Кабинет

Пример:

> Сделай личный кабинет соискателя по этим данным.

Создать: Applicant Cabinet.

## Важно

Не подтягивать Дарью, котокафе, БАНКИРРО, Артура и старые выводы в новые кейсы, если пользователь этого не сказал.
