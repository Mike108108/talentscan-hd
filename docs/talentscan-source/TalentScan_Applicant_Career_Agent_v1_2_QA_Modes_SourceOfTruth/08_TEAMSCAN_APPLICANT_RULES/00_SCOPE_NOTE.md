# TeamScan Applicant Scope Note v1.2

В Applicant Agent TeamScan не равен “совместимость вообще”. Это инструмент карьерной проверки среды и людей.

## Активные режимы

1. **Мой идеальный руководитель** — строится по карте самого соискателя.
2. **Стиль руководителя по анкете** — без бодиграфа руководителя, по описанию управления.
3. **Конкретный руководитель** — если есть данные рождения / бодиграф руководителя.
4. **Команда / компания как среда** — только если есть структурированные данные; без них не называть это композитом.

## Главная рамка

Соискатель читает это для себя:

- стоит ли идти к этому руководителю;
- какие условия уточнить;
- какой стиль управления поможет раскрыться;
- где риски давления, мутной роли, ответственности без полномочий;
- что зафиксировать до “да”.

## Нельзя

- выдавать compatibility как судьбоносный приговор;
- писать “работать нельзя” без условий;
- смешивать любовные, дружеские, семейные и HR-сценарии;
- использовать старый multi-context compatibility pack как активную основу;
- делать вид, что есть композит с командой/компанией, если нет данных.
