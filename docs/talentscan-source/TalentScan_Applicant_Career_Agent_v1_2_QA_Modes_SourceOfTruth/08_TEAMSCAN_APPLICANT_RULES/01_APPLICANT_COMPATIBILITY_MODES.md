# Applicant Compatibility Modes

## Mode 1 — Ideal leader

Input: applicant data only.

Output:
- ideal management style;
- unsuitable management style;
- questions for interview.

Do not call it compatibility.

## Mode 2 — Leader style by questionnaire

Input: applicant data + described leadership style.

Output:
- fit with management style;
- risk areas;
- questions and agreements.

Mark that it is not bodygraph composite.

## Mode 3 — Specific leader by bodygraph

Input: applicant data + leader bodygraph/data.

Output:
- Applicant TeamScan;
- decision process;
- what to ask;
- conditions;
- technical mechanics if requested.

## Mode 4 — Team/company environment

Input: applicant data + team/company structured data.

Output:
- working environment fit;
- red/yellow/green conditions;
- questions.

Only call it TeamScan if there is real second-party data.
