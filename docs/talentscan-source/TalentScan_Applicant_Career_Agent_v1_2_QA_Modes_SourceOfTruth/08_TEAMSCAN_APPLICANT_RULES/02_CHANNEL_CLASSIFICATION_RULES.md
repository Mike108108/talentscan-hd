# Channel Classification Rules

Keep strict classification.

## Dominant channel

One person has full channel. Other person has no gates from that channel.

Interpretation: one person brings a stable function that the other meets.

## Compromise channel

One person has full channel. Other person has one gate from the same channel.

Interpretation: sensitive zone. The second person is partially involved and may feel overwritten or pressured.

## Companion / same channel

Both have the same full channel.

Interpretation: shared function / similarity, not dominance.

## Electromagnetic channel

Each person has one different gate, together closing the channel.

Interpretation: joint function appears only in the relationship.

## For applicant TeamScan

Translate classification into applicant checks:

- What should I ask?
- What condition protects me?
- Where can I confuse attraction with sustainability?
- Where can responsibility and authority get mixed?
