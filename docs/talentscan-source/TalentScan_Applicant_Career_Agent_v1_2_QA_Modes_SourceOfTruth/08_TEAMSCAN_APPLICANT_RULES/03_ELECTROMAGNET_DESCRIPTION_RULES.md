# Electromagnet Description Rules

Do not write only: “You have electromagnet X–Y.”

Every electromagnet must include:

1. What gate/person A brings.
2. What gate/person B brings.
3. What each side may feel incomplete without the other.
4. What joint function appears together.
5. The “superpower” of the connection.
6. Risk/friction.
7. Applicant decision check.

## Applicant decision check examples

- What should the applicant clarify before accepting?
- Which conditions make this superpower useful?
- Which conditions turn it into pressure?
- Is this real role fit or only chemistry/activation?

## Example framing

“Электромагнит может дать ощущение сильного включения. Но Дарье важно проверить, есть ли под это включение реальные условия: роль, полномочия, ресурсы, график, команда и право на паузу.”
