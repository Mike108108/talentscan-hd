# Decision and Conditions Rules

Applicant TeamScan must include practical decision support.

Mandatory blocks:

1. How the applicant should decide.
2. What to ask the leader/employer.
3. Conditions to check before accepting.
4. Green/yellow/red flags.
5. Red flags for self-protection.
6. Decision matrix.
7. Protocol before accepting.

## Important

A strong connection is not enough. The applicant must check:

- role clarity;
- authority;
- responsibility;
- feedback style;
- schedule;
- workload;
- boundaries;
- trial period;
- written agreements;
- whether their core talent is truly invited.
