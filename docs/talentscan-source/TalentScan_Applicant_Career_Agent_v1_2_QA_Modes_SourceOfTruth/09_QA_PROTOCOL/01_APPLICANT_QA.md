# Applicant-Specific QA

Check every generated card:

## Does it answer applicant questions?

- What should I do?
- What should I ask employer?
- What conditions should I check?
- How should I decide?
- Where can I reveal myself?
- Where can I burn out?

## Does it avoid HR framing?

No:
- “Company should hire / not hire you.”
- “Candidate is suitable for employer.”
- “Compatibility verdict for hiring.”

Yes:
- “You should check...”
- “This role fits you if...”
- “Before accepting, clarify...”
