# Applicant TeamScan QA

Before delivering TeamScan, verify:

1. Is there enough second-person data?
2. If not, is it framed as ideal leader / environment check rather than composite?
3. If technical HD language is used, does every technical point lead to action?
4. Are channels classified correctly: dominant / compromise / companion / electromagnetic?
5. Are centers included if technical data is available?
6. Are electromagnets described through both gates and joint function?
7. Is the final conclusion a decision aid, not a verdict?
8. Are questions to employer included?
9. Are conditions and red flags included?
10. Does the card help the applicant decide whether to test/accept/refuse?
