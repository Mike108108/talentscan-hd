# QA — Talent Map Card

Проверить перед выдачей карты талантов.

## Обязательные блоки

- [ ] Hero с именем и формулой человека.
- [ ] Быстрый общий вывод.
- [ ] Показатели точности / полноты данных.
- [ ] Карта талантов в рабочем переводе.
- [ ] Сильные стороны.
- [ ] Риски / зоны перегруза.
- [ ] Подходящие профессиональные направления с процентами.
- [ ] Спорные / низкоподходящие направления с процентами.
- [ ] Возможные должности / форматы.
- [ ] Подходящая рабочая среда.
- [ ] Практические проверки / задания.
- [ ] Что добавить для точности.
- [ ] Итоговая рекомендация.

## Запреты

- [ ] Не показывать технические термины без запроса.
- [ ] Не использовать текущую роль, если пользователь её не дал.
- [ ] Не переносить вакансии из прошлых кейсов.
- [ ] Не выдавать профессию как приговор.

## Дизайн

- [ ] Dark/Darya style.
- [ ] Нет верхнего burger в карте талантов.
- [ ] Есть горизонтальная навигация.
- [ ] Мобильная адаптация не сломана.
- [ ] Якоря работают.
