# QA — Comparison Card

Проверить перед выдачей сравнения.

## Обязательные блоки

- [ ] Ясно указаны сравниваемые варианты.
- [ ] Есть таблица по ключевым критериям:
  - смысл;
  - деньги;
  - график;
  - нагрузка;
  - рост;
  - среда;
  - руководитель/команда, если есть данные;
  - риск выгорания;
  - условия раскрытия.
- [ ] Есть итог по каждому варианту.
- [ ] Есть рекомендация: что выбрать / что уточнить / что протестировать.
- [ ] Есть “если А, то…” сценарии.

## Запреты

- [ ] Не сравнивать варианты, если один из них не задан.
- [ ] Не подтягивать старые варианты.
- [ ] Не выбирать за человека без условий.

## UX

- [ ] Если это кабинет — есть возможность выбрать варианты.
- [ ] Есть кнопки назад / к деталям / сравнить все.
