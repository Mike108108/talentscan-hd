# QA — Working Environment Card

Проверить перед выдачей карты рабочей среды.

## Обязательные блоки

- [ ] Какой темп подходит.
- [ ] Какой стиль руководителя подходит.
- [ ] Какой формат обратной связи подходит.
- [ ] Какая команда подходит.
- [ ] Какие задачи раскрывают.
- [ ] Какие условия перегружают.
- [ ] Красные флаги среды.
- [ ] Вопросы работодателю.

## Важно

Если нет данных компании или команды, не называть это совместимостью с компанией. Называть: “проверка рабочей среды”.

## Запреты

- [ ] Не писать “компания подходит/не подходит”, если есть только описание вакансии.
- [ ] Не имитировать точность без данных.
