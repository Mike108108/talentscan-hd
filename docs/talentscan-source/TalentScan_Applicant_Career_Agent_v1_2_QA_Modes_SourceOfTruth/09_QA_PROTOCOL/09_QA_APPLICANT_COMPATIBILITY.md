# QA — Applicant Compatibility / Manager Fit

Проверить перед выдачей совместимости с руководителем / командой.

## Обязательные блоки

- [ ] Для кого карточка: для соискателя, а не работодателя.
- [ ] Быстрый вывод: можно смотреть / нужна пауза / красные условия.
- [ ] Что может быть сильным.
- [ ] Как человеку принимать решение.
- [ ] Что уточнить у работодателя / руководителя.
- [ ] Какие условия проверить до “да”.
- [ ] Какой стиль руководителя подходит.
- [ ] Какая роль рядом с этим руководителем подходит.
- [ ] Риски и красные флаги.
- [ ] Договорённости до выхода.
- [ ] Матрица “да / пауза / нет”.
- [ ] Итог.

## Если есть данные второго человека

- [ ] Можно делать точную рабочую связку.
- [ ] Но не использовать как приговор.
- [ ] По умолчанию клиентский язык без ДЧ-терминов.

## Если нет данных второго человека

- [ ] Не делать композит.
- [ ] Делать проверку стиля руководителя по анкете или проверку рабочей среды.

## Запреты

- [ ] Не использовать старые love/friendship/family contexts.
- [ ] Не писать “не работайте вместе” без условий.
- [ ] Не показывать техническую механику без запроса.
- [ ] Не подтягивать Артура как руководителя, если он не указан.
