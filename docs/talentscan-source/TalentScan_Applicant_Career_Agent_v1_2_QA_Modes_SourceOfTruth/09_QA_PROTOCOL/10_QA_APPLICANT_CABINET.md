# QA — Applicant Cabinet

Проверить перед выдачей личного кабинета.

## Структура

- [ ] Обзор.
- [ ] Карта талантов.
- [ ] Профориентация.
- [ ] Моя текущая роль.
- [ ] Вакансии.
- [ ] Сравнение.
- [ ] Рабочая среда.
- [ ] Совместимости.
- [ ] Интервью.
- [ ] План.
- [ ] Данные.

## UX

- [ ] Быстрые карточки на верхнем уровне.
- [ ] Кнопки “читать подробнее” для детальных карточек.
- [ ] Вложенные страницы открываются корректно.
- [ ] Везде, где нужно, есть “назад”.
- [ ] В текущей роли есть “изменить”.
- [ ] В вакансиях есть “добавить вакансию”.
- [ ] В совместимостях есть “добавить совместимость”.
- [ ] Сравнение позволяет выбрать варианты.

## Технический QA

- [ ] Все `data-go` ведут на существующие экраны.
- [ ] Нет битых якорей.
- [ ] Встроенные iframe/srcdoc открываются.
- [ ] Нет старой карточки совместимости, если должна быть новая.
- [ ] Нет старой вакансии/роли, если пользователь просил заменить.
