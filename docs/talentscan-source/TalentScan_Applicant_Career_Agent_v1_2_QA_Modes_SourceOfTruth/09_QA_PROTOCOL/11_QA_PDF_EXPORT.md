# QA — PDF export

PDF делать только по прямой просьбе пользователя.

## Перед PDF

- [ ] Уточнить, какие страницы нужны, если пользователь не перечислил.
- [ ] Использовать готовый HTML как источник.
- [ ] Подготовить print-friendly CSS: убрать навигацию, sticky меню, лишние кнопки.
- [ ] Сохранить dark дизайн и читаемость.

## Проверить после генерации

- [ ] PDF открывается.
- [ ] Кириллица отображается корректно.
- [ ] Первый экран / hero не сломан.
- [ ] Таблицы не обрезаны критично.
- [ ] Основные блоки есть.
- [ ] Нет пустых страниц в начале.
- [ ] Если много PDF — сделать ZIP.

## Не делать

- [ ] Не делать PDF по умолчанию.
- [ ] Не делать PDF вместо HTML, если пользователь просит HTML.
- [ ] Не терять исходные HTML.
