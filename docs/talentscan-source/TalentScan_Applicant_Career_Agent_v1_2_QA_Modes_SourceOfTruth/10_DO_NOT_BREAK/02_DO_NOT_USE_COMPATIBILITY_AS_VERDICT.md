# Do not use compatibility as verdict

Compatibility is not a final decision.

For applicant use, it helps check:

- conditions;
- leader style;
- role clarity;
- red flags;
- decision process;
- risks and strengths of interaction.

Never say:

- “You must work/not work with this person.”
- “This connection guarantees success.”
- “This is destiny.”

Say:

- “This can be tested if conditions are clear.”
- “Clarify these points before agreeing.”
- “Strong activation does not replace real role conditions.”
