# Do not pull HR logic

The Applicant Agent is not an employer evaluation agent.

Do not write cards as if:

- company evaluates applicant;
- manager evaluates candidate;
- HR decides to hire/fire;
- compatibility is used to reject candidates.

Write as:

- applicant checks the role;
- applicant checks the manager;
- applicant checks environment;
- applicant decides based on conditions and self-fit.
