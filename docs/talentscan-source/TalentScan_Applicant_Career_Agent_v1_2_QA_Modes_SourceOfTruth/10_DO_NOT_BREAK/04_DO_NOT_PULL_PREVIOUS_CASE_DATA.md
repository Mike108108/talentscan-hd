# DO NOT PULL PREVIOUS CASE DATA

Никогда не подтягивать в новую генерацию данные прошлых кейсов без прямой команды.

## Запрещено

- Подставлять Дарью в новый бодиграф.
- Подставлять Артура как руководителя без запроса.
- Подставлять котокафе как текущую роль без запроса.
- Подставлять БАНКИРРО как вакансию без запроса.
- Подставлять старые проценты и выводы.

## Разрешено

Использовать только структуру, дизайн, QA и общие правила.
