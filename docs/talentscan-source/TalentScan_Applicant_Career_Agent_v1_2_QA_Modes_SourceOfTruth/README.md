# TalentScan Applicant / Career Agent v1.2

Это чистая рабочая база агента ТалентСкан для соискателя / человека, который выбирает работу, текущую роль, вакансию, руководителя и среду.

## Главная рамка

Агент не оценивает человека для работодателя. Агент помогает человеку проверить:

- какая работа ему подходит;
- раскрывает ли его текущая роль;
- стоит ли рассматривать конкретную вакансию;
- какие условия уточнить у работодателя;
- какой руководитель и среда ему подходят;
- где есть риск выгорания, давления, мутной роли или ответственности без полномочий;
- как принять карьерное решение и что сделать дальше.

## Активные эталоны v1.2

1. `05_ACTIVE_HTML_ETALON/ACTIVE_DARYA_TALENT_MAP_DARK_V1_1_NO_BURGER_LOWFIT.html` — главный дизайн и базовая структура карты талантов.
2. `11_HTML_EXAMPLES/EXAMPLE_DARYA_CURRENT_ROLE_KOTOKAFE_DARK_V1_0.html` — эталон карточки текущей роли.
3. `11_HTML_EXAMPLES/EXAMPLE_MIKHAIL_ONESUMMER_VACANCY_ASSESSMENT_V2_SELF_PRESENTATION.html` — активный эталон карточки соответствия вакансии, включая блок “Как преподнести себя на собеседовании”.
   - `11_HTML_EXAMPLES/EXAMPLE_DARYA_VACANCY_BANKIRRO_CLIENT_MANAGER_DARK_V1_0.html` — дополнительный пример карточки вакансии, не главный эталон.
4. `11_HTML_EXAMPLES/EXAMPLE_ARTUR_DARYA_APPLICANT_COMPATIBILITY_PLAIN_LANGUAGE_V1_0.html` — эталон совместимости с руководителем для соискателя простым языком.
5. `11_HTML_EXAMPLES/EXAMPLE_DARYA_APPLICANT_CABINET_V1_3_MULTILEVEL.html` — эталон многоуровневого кабинета соискателя.

## Статус

Эта версия годится для запуска отдельного GPT-агента и тестирования на новых людях. Она не финальная production-база: шаблоны и сценарии дальше можно расширять.
