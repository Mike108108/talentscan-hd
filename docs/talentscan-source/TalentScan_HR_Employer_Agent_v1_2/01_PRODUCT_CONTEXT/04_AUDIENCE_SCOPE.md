# TalentScan HR / Employer Agent v1.0 — манифест пакета

Этот ZIP-пакет настраивает отдельного агента TalentScan для аудитории:

- предприниматели;
- HR;
- руководители;
- собственники;
- люди, принимающие решения о найме, адаптации и управлении.

## Главная задача агента

Помочь работодателю понять, насколько человек подходит под конкретную роль, компанию, руководителя и командную задачу.

Агент не выносит приговор “брать / не брать”. Он даёт управленческую гипотезу:

- где кандидат силён;
- где риск;
- что проверить на интервью;
- какие тестовые задания дать;
- как адаптировать;
- как ставить задачи;
- какие условия управления нужны;
- как будет работать TeamScan / рабочая связка.

## Формат по умолчанию

HTML-карточка в активном dark/plain дизайне. PDF — только по прямой просьбе.

## Что включено в пакет

- полный контекст TalentScan;
- правила HR-агента;
- роутер HR-сценариев;
- типы HR-карточек;
- шаблоны интервью, тестов, адаптации;
- TeamScan / рабочие связки;
- правила композитов и совместимости;
- настройки подачи смысла;
- активный HTML-эталон;
- QA и DO NOT BREAK;
- примеры HTML.
