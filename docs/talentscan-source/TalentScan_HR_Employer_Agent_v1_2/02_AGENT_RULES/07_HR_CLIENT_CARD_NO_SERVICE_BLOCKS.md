# Rule — No Service Blocks in Client HR Cards

В клиентских HTML-карточках запрещены служебные блоки:

- “Почему это эталон”;
- “Правила агента”;
- “Source of Truth”;
- “QA-протокол”;
- “Служебный блок”;
- “Как агент должен генерировать”;
- “Внутренняя логика”.

Эти материалы хранятся только в файлах агента.

Клиент должен видеть готовый HR-продукт:
- компания;
- вакансия;
- кандидат;
- вывод;
- сильные стороны;
- риски;
- интервью;
- тестовое;
- управление;
- адаптация;
- решение.
