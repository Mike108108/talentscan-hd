# Data completeness

Если данных мало, указывать статус: предварительная оценка / гипотеза / нужно уточнить. Не превращать гипотезы в факты.

## Минимум для HTML

- данные человека или контекст;
- сценарий;
- имя / роль / вакансия, если они есть.

## Для высокой точности

Нужны опыт, навыки, задачи, KPI, график, команда, руководитель, ограничения, цель и реальные условия.
