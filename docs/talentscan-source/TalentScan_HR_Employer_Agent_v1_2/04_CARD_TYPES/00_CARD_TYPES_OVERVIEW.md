# HR Card Types v1.1

## 1. Кандидат под конкретную роль

Когда использовать:
- есть один кандидат;
- есть вакансия/роль;
- нужно понять, вести ли кандидата дальше.

Активный эталон:
`11_HTML_EXAMPLES/ACTIVE_HR_CLIENT_ETALONS_v1_1/ETALON_01_HR_CANDIDATE_FOR_ROLE_MARINA_ONESUMMER_SALES_v1_2_1.html`

## 2. Сравнение кандидатов на одну роль

Когда использовать:
- есть 2–5 кандидатов;
- есть одна роль;
- нужно выбрать приоритет, финалистов, резерв, зоны проверки.

Активный эталон:
`11_HTML_EXAMPLES/ACTIVE_HR_CLIENT_ETALONS_v1_1/ETALON_02_HR_CANDIDATE_COMPARISON_MARINA_ZARINA_ONESUMMER_SALES_v1_1.html`

## 3. Рабочая связка руководитель × кандидат

Когда использовать:
- есть руководитель;
- есть кандидат/сотрудник;
- нужно понять управление, коммуникацию, риски, адаптацию.

Активный эталон:
`11_HTML_EXAMPLES/ACTIVE_HR_CLIENT_ETALONS_v1_1/ETALON_03_HR_TEAMSCAN_ARTUR_MARINA_ONESUMMER_SALES_v1_0.html`

## 4. Профиль роли / вакансии

Когда использовать:
- есть компания и вакансия;
- кандидата ещё нет;
- нужно определить, какие качества искать и как проверять.

## 5. Предварительная HR-карта кандидата

Когда использовать:
- есть только кандидат;
- нет роли/компании;
- нужно дать предварительный рабочий профиль, без финального процента fit под роль.


## 4. Person Talent Map / Карта талантов человека

Базовая карта человека без обязательной привязки к вакансии. Активный эталон: `11_HTML_EXAMPLES/ACTIVE_HR_CLIENT_ETALONS_v1_2/ETALON_00_HR_PERSON_TALENT_MAP_DARYA_v1_1.html`.
