# Активный HTML-эталон дизайна

Активный эталон всех клиентских HTML-карточек TalentScan:

`talentscan_olga_career_profile_plain_dark_v2_mobile_fixed_menu.html`

## Обязательные свойства дизайна

- тёмный plain/dark стиль;
- hero-блок;
- карточная структура;
- адаптивные сетки;
- fixed/sticky мобильная верхняя панель;
- бургер-меню на телефоне;
- горизонтальная навигация по разделам;
- рабочие ссылки по блокам;
- кнопка `↑ Вернуться в начало`;
- корректная работа на ПК и телефоне через click/tap/touch;
- автономный HTML: CSS и JS внутри файла, без внешних библиотек.

## Что можно менять

Можно менять:

- имя;
- сценарий;
- названия разделов;
- контент;
- проценты;
- формулировки;
- навигационные пункты под конкретный тип карточки.

## Что нельзя менять без прямой команды

Нельзя менять:

- визуальную систему;
- мобильную механику меню;
- принцип fixed/sticky-навигации;
- структуру hero → nav → sections → final → back-to-top;
- client-safe язык;
- HTML-first формат.
