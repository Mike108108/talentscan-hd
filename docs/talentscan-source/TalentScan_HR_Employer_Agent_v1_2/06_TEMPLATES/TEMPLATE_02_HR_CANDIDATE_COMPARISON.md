# Template — HR Candidate Comparison v1.1

## Назначение

Сравнить 2–5 кандидатов на одну роль.

## Обязательная структура HTML

1. Hero:
   - роль;
   - компания;
   - кандидаты;
   - быстрые проценты;
   - предварительный приоритет.

2. Быстрый HR-вывод:
   - кого вести первым;
   - кого оставить финалистом/резервом;
   - какая развилка в решении.

3. Краткий контекст компании/вакансии:
   - только критерии, которые влияют на выбор.

4. Сравнительная матрица:
   - критерий;
   - вес для роли;
   - кандидат 1;
   - кандидат 2;
   - комментарий для HR.

5. Профили кандидатов:
   - как каждый будет работать;
   - где сильнее;
   - где риск.

6. Сильные стороны каждого.

7. Риски каждого.

8. Интервью:
   - общие вопросы;
   - разводящие вопросы.

9. Тестовое:
   - одинаковые задания для честного сравнения.

10. Решение:
   - если нужен один кандидат;
   - если можно вести двоих;
   - кого держать резервом;
   - что проверить перед оффером.

## Нельзя

- сравнивать “кто лучше вообще”;
- делать выбор только по симпатии;
- перегружать компанию/вакансию тем, что HR и так знает.
