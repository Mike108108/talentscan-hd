# Template — HR интервью и тестовые задания

Интервью и тесты должны проверять реальные риски роли, а не быть общими вопросами “расскажите о себе”.

## Интервью

- вопросы по ключевым задачам роли;
- вопросы по рискам кандидата;
- вопросы по конфликтам и обратной связи;
- вопросы по самостоятельности и срокам;
- вопросы по KPI и рабочему ритму.

## Тесты

- практический кейс роли;
- коммуникационный кейс;
- кейс по приоритетам;
- кейс по CRM/таблице/процессу;
- короткое задание на 30–60 минут.
