# Template — HR Person Talent Map

## Когда использовать

Используй этот шаблон, если работодатель/HR просит:
- карту талантов кандидата;
- карту талантов сотрудника;
- рабочий профиль человека;
- понять, какие роли подходят человеку;
- проанализировать человека без конкретной вакансии.

## Структура результата

### 1. Hero
- имя человека;
- статус: кандидат / сотрудник / руководитель / без роли;
- краткая формула рабочей природы;
- лучший рабочий формат;
- главный талант;
- главный риск.

### 2. Общий вывод
- куда человеку лучше смотреть в работе;
- в каких задачах он полезен компании;
- какие роли потенциально подходят.

### 3. Показатели оценки
- полнота данных;
- уверенность оценки;
- количество направлений для проверки;
- что нужно дозаполнить.

### 4. Таланты в рабочем переводе
5–9 талантов в формате:
- суть;
- как проявляется в работе;
- где использовать;
- риск / как проверить.

### 5. Сильные стороны
Список рабочих сильных сторон без общих комплиментов.

### 6. Риски
Риски как условия, где эффективность падает.

### 7. Подходящие направления
Таблица: направление / процент / почему подходит / что проверить.

### 8. Спорные направления
Таблица: направление / низкое соответствие / почему спорно / когда допустимо.

### 9. Должности
Конкретные роли, которые стоит рассмотреть.

### 10. Рабочая среда
Подходящие и неподходящие условия.

### 11. Проверка
Мини-тесты и практические задания, чтобы подтвердить гипотезы.

### 12. Итоговая рекомендация
Короткий управленческий вывод для HR/работодателя.
