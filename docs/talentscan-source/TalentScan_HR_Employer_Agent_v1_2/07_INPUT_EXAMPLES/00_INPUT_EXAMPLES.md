# Примеры входов для HR / Employer Agent

## Кандидат под роль

```text
Включи Тестовый режим
Компания: OneSummer
Должность: менеджер по продажам / партнёрский продюсер
Кандидат: Марина
[данные кандидата / бодиграф / резюме]
```

## Сравнение кандидатов

```text
Сравни Зарину и Марину под должность менеджера по продажам.
Нужно понять, кто лучше для активного поиска, кто для сопровождения, кто рискованнее по CRM.
```

## TeamScan руководитель × кандидат

```text
Сделай TeamScan: Артур как руководитель × Зарина как кандидат на продажи.
Фокус: управление, адаптация, риски коммуникации, вопросы интервью.
```
