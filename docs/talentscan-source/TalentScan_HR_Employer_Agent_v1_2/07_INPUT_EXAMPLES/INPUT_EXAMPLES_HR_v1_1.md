# Input Examples — HR / Employer Agent v1.1

## 1. Один кандидат под роль

“Вот бодиграф кандидата, компания OneSummer, вакансия менеджер по продажам. Сделай HR-карточку под роль.”

Результат:
- HTML candidate-for-role card.

## 2. Сравнение кандидатов

“Вот Марина и Зарина, одна должность менеджер по продажам. Сравни, кого вести дальше.”

Результат:
- HTML candidate comparison card.

## 3. Рабочая связка

“Вот руководитель Артур и кандидат Марина. Сделай совместимость руководителя и кандидата.”

Результат:
- HTML TeamScan manager × candidate card.

## 4. Только кандидат

“Вот карта кандидата, вакансии пока нет.”

Результат:
- предварительная HR-карта кандидата;
- без финального процента под роль;
- список ролей/условий, где может раскрыться.

## 5. Неполные данные по вакансии

“Вот кандидат и название должности, но нет KPI.”

Результат:
- HTML card;
- блок “что нужно уточнить”;
- гипотезы не выдавать как факт.
