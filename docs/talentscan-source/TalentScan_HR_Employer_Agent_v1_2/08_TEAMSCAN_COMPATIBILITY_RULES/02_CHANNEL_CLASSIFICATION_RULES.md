# Правила классификации каналов для TeamScan

Перед выдачей анализа проверить каждый канал.

## 1. Доминантный канал

Полный канал у человека А + у человека Б нет ни одних ворот этого канала.

Это доминанта человека А.

## 2. Компромиссный канал

Полный канал у человека А + у человека Б есть одни ворота этого же канала.

Это компромисс, а не доминанта.

## 3. Одинаковый / общий канал

Полный канал есть у обоих.

Это общий / одинаковый / компаньонский канал, но не доминанта.

## 4. Электромагнит

У одного человека одни ворота, у второго вторые ворота, вместе закрывают канал.

Это новая совместная функция на двоих.

## 5. Неуверенные данные

Если данные считаны по скрину и не уверены на 100%, писать:

- “по видимым данным”;
- “гипотеза по скрину”;
- “для финальной версии нужно сверить список каналов/ворот”.

Не писать категорично, если данные не подтверждены.
