# Правило раскрытия электромагнитов

Нельзя писать только “у вас электромагнит X–Y”.

Каждый электромагнит раскрывать через формулу:

1. У человека А есть часть темы.
2. У человека Б есть дополняющая часть темы.
3. По отдельности каждая сторона неполна.
4. Вместе закрывается новая совместная функция.
5. Описать суперспособность связки.
6. Добавить риск.
7. Дать жизненные / рабочие примеры.

## Клиентский шаблон

```text
Совместная функция: [название]

Суть:
Между людьми включается тема ...

Что приносит первый человек:
...

Что приносит второй человек:
...

Почему по отдельности неполно:
...

Суперспособность связки:
Вместе они могут ...

Риск:
...

Что делать:
...
```
