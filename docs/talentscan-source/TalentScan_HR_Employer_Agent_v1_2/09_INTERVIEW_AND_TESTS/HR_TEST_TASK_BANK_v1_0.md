# HR Test Task Bank v1.0

## Тест 1. Ролевая продажа

Дать кандидату описание продукта и клиента. Попросить провести мини-разговор:
- выявить потребность;
- объяснить ценность;
- обработать сомнение;
- предложить следующий шаг.

Что смотреть:
- контакт;
- вопросы;
- аргументацию;
- структуру;
- дожим без давления.

## Тест 2. Follow-up

Ситуация: клиент сказал “интересно, я подумаю”.

Попросить кандидата составить:
- сообщение через 1 день;
- звонок через 3 дня;
- аргумент/ценность для возвращения к решению.

Что смотреть:
- зрелость сопровождения;
- отсутствие навязчивости;
- умение двигать сделку.

## Тест 3. CRM-кейс

Дать 6 лидов на разных стадиях.

Попросить:
- расставить приоритеты;
- описать следующие действия;
- показать, что будет внесено в CRM.

Что смотреть:
- структурность;
- дисциплину;
- понимание воронки;
- способность работать удалённо.

## Тест 4. Продажа OneSummer преподавателю

Сценарий: йога-преподаватель думает провести тур на Шри-Ланке.

Попросить кандидата:
- выявить формат группы;
- показать ценность локации;
- объяснить сопровождение;
- довести до следующего шага.

Что смотреть:
- понимание экспертной аудитории;
- доверие;
- партнёрский тон;
- конкретность следующего шага.
