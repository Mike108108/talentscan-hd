# QA Protocol Folder

Главный активный QA-файл:

`00_ACTIVE_HR_EMPLOYER_QA_PROTOCOL_v1_1.md`

Перед выдачей любой HTML-карточки агент обязан проверить:

- соответствие типу карточки;
- клиентский HR-язык;
- отсутствие служебных блоков;
- отсутствие technical HD/socionics terms;
- наличие быстрого вывода;
- наличие интервью/тестового/решения;
- разделение фактов, гипотез и данных, которые нужно уточнить;
- независимость нового кейса от прошлого;
- мобильную адаптацию и отсутствие наложений.
