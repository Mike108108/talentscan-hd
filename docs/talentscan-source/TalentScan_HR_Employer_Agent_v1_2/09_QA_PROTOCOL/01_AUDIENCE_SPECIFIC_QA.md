# QA HR / Employer Agent

Перед выдачей проверить:

1. Компания, роль, кандидат не перепутаны.
2. Не подтянут прошлый кандидат или прошлая вакансия.
3. Результат не звучит как окончательный приговор найма.
4. Есть блок “что проверить”.
5. Есть вопросы интервью, если это оценка кандидата.
6. Есть тестовые задания, если это кандидат под роль.
7. Если TeamScan — есть договорённости, риски управления и адаптация.
8. В клиентской версии нет технической терминологии.
9. HTML сделан в активном dark/plain дизайне.
10. Мобильное меню, tap/touch, кнопки и “Вернуться в начало” работают.
