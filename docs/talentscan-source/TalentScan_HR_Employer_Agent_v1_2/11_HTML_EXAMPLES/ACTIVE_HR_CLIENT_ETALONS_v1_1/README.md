# Active HR Client Etalons v1.1

Эта папка содержит активные клиентские HTML-эталоны для HR/Employer Agent.

## Файлы

1. `ETALON_01_HR_CANDIDATE_FOR_ROLE_MARINA_ONESUMMER_SALES_v1_2_1.html`
   - кандидат под роль;
   - Марина × OneSummer × менеджер по продажам.

2. `ETALON_02_HR_CANDIDATE_COMPARISON_MARINA_ZARINA_ONESUMMER_SALES_v1_1.html`
   - сравнение кандидатов на одну роль;
   - Марина vs Зарина.

3. `ETALON_03_HR_TEAMSCAN_ARTUR_MARINA_ONESUMMER_SALES_v1_0.html`
   - рабочая связка руководитель × кандидат;
   - Артур × Марина.

## Как использовать

- Использовать как эталоны структуры, клиентского HR-языка и UX.
- Не копировать персональные выводы на новых людей.
- Не добавлять в клиентские карточки служебные блоки агента.
- Сохранять dark/plain style и мобильную адаптацию.
