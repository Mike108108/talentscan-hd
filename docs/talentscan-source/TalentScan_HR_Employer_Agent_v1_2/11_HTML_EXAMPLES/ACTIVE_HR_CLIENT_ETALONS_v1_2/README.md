# Active HR Client Etalons v1.2

Активные клиентские HTML-эталоны HR / Employer Agent.

## 00. Карта талантов человека
`ETALON_00_HR_PERSON_TALENT_MAP_DARYA_v1_1.html`

Использовать как базовый эталон карточки человека без обязательной привязки к вакансии: кандидат, сотрудник, руководитель, человек без роли.

Обязательная логика:
- общий вывод;
- показатели оценки;
- таланты в рабочем переводе;
- сильные стороны;
- риски;
- подходящие направления;
- спорные направления;
- должности;
- рабочая среда;
- проверка;
- итоговая рекомендация.

## 01. Кандидат под конкретную роль
`ETALON_01_HR_CANDIDATE_FOR_ROLE_MARINA_ONESUMMER_SALES_v1_2_1.html`

Использовать, когда есть кандидат + конкретная роль / вакансия / компания.

## 02. Сравнение кандидатов
`ETALON_02_HR_CANDIDATE_COMPARISON_MARINA_ZARINA_ONESUMMER_SALES_v1_1.html`

Использовать, когда есть 2+ кандидата на одну роль.

## 03. TeamScan руководитель × кандидат/сотрудник
`ETALON_03_HR_TEAMSCAN_ARTUR_MARINA_ONESUMMER_SALES_v1_0.html`

Использовать для рабочей связки руководитель × кандидат/сотрудник.

## Правило клиентского слоя

Клиентские карточки не содержат служебных блоков агента, внутренних методологических комментариев и технических терминов ДЧ/соционики. Все выводы переводятся в рабочий язык HR/работодателя.
