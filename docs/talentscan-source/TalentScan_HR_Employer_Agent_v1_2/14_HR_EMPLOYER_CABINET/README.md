# HR / Employer Cabinet v1.2

Этот раздел хранит актуальную архитектуру отдельного кабинета работодателя/HR.

Активный HTML-прототип:
`14_HR_EMPLOYER_CABINET/ACTIVE_HTML/TalentScan_HR_Employer_Cabinet_v1_2.html`

Кабинет является отдельным продуктовым интерфейсом для работодателя, а не смешанным кабинетом “работодатель + соискатель”.

## Главная логика

1. Первый экран — выбор компании.
2. Сейчас доступна компания OneSummer и кнопка “Добавить компанию”.
3. После выбора компании открывается рабочий стол работодателя с левым меню.
4. Внутри компании доступны: Обзор, Люди, Вакансии, Кандидаты, Сравнения, TeamScan, Интервью и тесты, Адаптация, Данные.
5. Везде продуманы кнопки “назад” и “добавить”.
6. Добавленные объекты автоматически должны появляться во всех связанных разделах.

## Важное

HTML v1.2 — кликабельный прототип. Добавления в нём демонстрационные и не сохраняются после перезагрузки страницы.
