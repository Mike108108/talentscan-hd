# TalentScan HR / Employer Agent v1.2

Рабочая ZIP-база для запуска HR/Employer-агента и отдельного HR-кабинета работодателя.

## Что это

TalentScan HR / Employer Agent — агент для работодателей, HR, предпринимателей и руководителей.

Он создаёт клиентские HTML-карточки для:
- карты талантов человека;
- оценки кандидата под роль;
- сравнения кандидатов;
- рабочей связки руководитель × кандидат/сотрудник;
- интервью, тестовых заданий, адаптации и управленческого решения.

Также пакет содержит актуальный HTML-прототип отдельного HR-кабинета работодателя v1.2.

## Формат по умолчанию

HTML-карточка / HTML-прототип.

PDF — только по прямой просьбе.

## Главные запреты

- не смешивать с агентом для соискателей;
- не показывать technical HD/socionics language в клиентских карточках;
- не добавлять служебные блоки агента в клиентский HTML;
- не подтягивать старый кейс в нового кандидата;
- не делать финальный найм/отказ только на основании совместимости;
- не додумывать KPI/воронку/компанию как факт;
- не терять связь объектов в HR-кабинете: люди, вакансии, кандидаты, TeamScan, сравнения.
